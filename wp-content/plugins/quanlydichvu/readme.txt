== Mô tả ==
Plugin giúp quản lý và hiển thị dịch vụ chăm sóc sau sinh. Bạn có thể thêm dịch vụ mới, chỉnh sửa, và hiển thị bằng shortcode.

== Cách sử dụng ==
1. Kích hoạt plugin.
2. Vào "Dịch vụ" -> "Thêm mới".
3. Dán shortcode [dichvu_list] vào trang bất kỳ để hiển thị dịch vụ.
